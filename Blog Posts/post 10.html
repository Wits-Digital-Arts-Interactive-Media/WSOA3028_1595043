Benefits and Challenges of responsive design

From what I had gathered from the lectures, it is not surprising that when designing a website or webpage or anything involving the web/internet that a designer must take into consideration that any device can handle it. However, this is much easier said than done. It is correct that it allows much easier client usability and navigation and is more convenient for them as they can access it with any device. Yet this then requires additional research into programming the graphics for the website to making it automatically adjust resolution and element placement. This can take time and large adjustments of code as to achieve the same effect that the website or webpage desires psychologically can be limited as now with every hardware, what it can do is different from any other hardware. So, this must then be programmed so that the code is “automatic” in adjustment while changing certain elements to still maintain usability and navigation.

Reflection

In the end, the article shows that internet access is not that simple and is much more complex than one would expect. From what I have gathered, it is clear that research into it Is of the utmost importance as many problems can be found surrounding the topic.
